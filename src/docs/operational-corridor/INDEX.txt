REFERÊNCIA RÁPIDA DE ENDPOINTS
================================================================================

GET /health
  └─ Status do gateway
  └─ Resposta: { status: "healthy", correlationId: "..." }

GET /entities/{entity}/list?limit=50&page=0&sort=-created_date
  └─ Listar entidades com paginação
  └─ Resposta: { data: [...], hasMore: true, page: 0, limit: 50 }

GET /entities/{entity}/export (with Idempotency-Key header)
  └─ Exportar em CSV (idempotente)
  └─ Resposta: CSV file (mesmo conteúdo se repetir com mesma key)

GET /sync/status
  └─ Status de sincronização por entidade
  └─ Resposta: { TargetFlare: { healthy: true, ... }, ... }

GET /automations/status
  └─ Status de todas automations
  └─ Resposta: [ { name: "...", status: "active", ... }, ... ]

POST /github/webhook
  └─ Listener para eventos GitHub
  └─ Resposta: { status: "accepted", correlationId: "..." }
